A Simple Calculator that does the following 2 operand functions:====================================================
	Addition (+)=======================================================
	Subtraction (-)====================================
	Multiplication (*)===================================
	Division (/)==================================
	Log (l)================================================
	Modulus(%)==================================
	Power(^)=========================

The format is OPERAND1 OPERATOR OPERAND2
where OPERAND1 and OPERAND2 are integers
